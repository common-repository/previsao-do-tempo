Plugin que exibe a previsão do tempo para o Brasil

== Description ==

O plugin da Climatempo permite maior versatilidade em seu site ou blog. Você poderá contar com a previsão do tempo de alta qualidade inteiramente grátis.
Com ele você terá um selo com a previsão do tempo para as cidades a sua escolha.

== Installation ==

Manual:
Após descompactar o arquivo previsao-do-tempo.zip na pasta /wp-content/plugins/, abrir o gerenciador de plugins no admin do seu wordpress, 
ativar o plugin. Depois disso, atualize a página (F5), vá nos widgets, adicione-o, escolha as cidades que deseja exibir a previsão, tamanho, cor do widget e salve.

Painel:
Após fazer o download e instalar, atualize a página (F5), vá nos widgets, adicione-o, escolha as cidades que deseja exibir a previsão, tamanho, cor do widget e salve.
